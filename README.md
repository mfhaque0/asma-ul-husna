# 🌙 Asma Ul Husna – The 99 Beautiful Names of Allah

A beautiful and elegant single-page website displaying the 99 Names of Allah (Asma Ul Husna) with Arabic text, English transliteration, and meanings.

## ✨ Features

- Beautiful glassmorphism design
- Elegant typography using Google Fonts
- Responsive mobile-friendly layout
- Interactive cards with expand/collapse animation
- Smooth scrolling experience
- Fully contained in a single HTML file

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
